GiftMe

Purpose

GiftMe is a gift exchange web application. In GiftMe individuals will be able to form a public or private exchange group and preform an online white elephant gift exchange. This project is still in the early stages of development with an expected release for late September.

Frameworks

    The creation of this project utilizes the MEAN stack.
  • The backend of this projcet is being written in Node-Express with MongoDB as the database and Mongoose as the ORM.
  • The Fronend of this application is being written in Angular.

Current Features

Login

On a login attempt, the client first provides input validation before formatting and serializing the request to the backend. Using proper password handling techniques, the backend handles the incoming request from the Client and verifies the sent credentials. In a successful login a session token is generated and stored in local storage to the client and the client is sent to their protected home page.

Registration

On a registration attempt, the client first provides input validation before formatting and serializing the request to the backend. On receiving the registration attempt the UserName must be unique and is checked against database records. If successful the password is hashed and salted before being stored as a record in the database. The client is then redirected to login.
