CM Site

Purpose

The CM Site is my personal website that I started working on in the Summer of 2019. This site was built to give me more practical experince in developing the UI of a website and as a log of my experince and projects.


Framework and Languages

This site is a Single Page Application (SPA) built out with the Angular and Node-Express framework. The backend is written in vanilla JS while the frontend this site was built with typescript, SCSS and HTML.

Functionality

    The following is the current list of functionality this site provides;
  • Rendering of static images, HTML, and SCSS into the DOM through the Rendering and JavaScript Engine
  • CORS request to the Backend API Route of a message subject and body thorugh the Contact form. The backend will then email a valid request to a mail server using SMTP.

Dependencies

  • MD Bootstrap for Angular
  • Flash Messages
  • Body Parser
  • CORS
